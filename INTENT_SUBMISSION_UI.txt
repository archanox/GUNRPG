✅ Real-time selection display
✅ Submit intents to combat system via POST /sessions/{id}/intent
✅ Validation and error handling
✅ Success/error messages
✅ Returns to combat screen after submission

FLOW:
1. From Combat Session → Select "SUBMIT INTENTS"
2. Choose action categories and set each intent
3. Review selections in "CURRENT SELECTIONS" panel
4. Select "CONFIRM & SUBMIT"
5. Intents sent to API
6. Return to Combat Session screen
7. Use "ADVANCE TURN" to resolve combat
